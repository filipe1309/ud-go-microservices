## :cd: Installation

```sh
git clone git@github.com:filipe1309/ud-go-microservices.git
```

```sh
cd ud-go-microservices
```

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## :runner: Running

Start all containers:
```sh
make up_build
```
> Stop with `make down`

Start front end app:
```sh
make start # start front-end
```
> Stop with `make stop`

> Access http://localhost
