
### 69. Generating the gRPC code from the command line

Install protoc:
```sh
brew install protobuf
protoc --version
```

Inside `logger-service/logs`, run:
```sh
protoc --go_out=. --go_opt=paths=source_relative --go-grpc_out=. --go-grpc_opt=paths=source_relative logs.proto
```
